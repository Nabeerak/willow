# API Contracts

## WebSocket Protocol

**URL**: `ws://{host}/voice/{session_id}`

### Frame Types

| Frame | Direction | Content |
|-------|-----------|---------|
| Binary | Client → Server | Raw audio bytes (16kHz, 16-bit PCM, mono) |
| Text | Client → Server | JSON control message |
| Text | Server → Client | JSON control command |

### Client → Server Control Messages

```json
{"type": "preflight_start"}
{"type": "preflight_end"}
```

- `preflight_start`: Browser audio warmup begun (T028). Tier 4 is suppressed until `preflight_end`.
- `preflight_end`: Warmup complete. Full pipeline active.

### Server → Client Control Commands

```json
{"type": "flush_audio_buffer", "fade_duration_ms": 7}
```

- `flush_audio_buffer`: Tier 4 fired; browser should fade out in-flight audio (T025, FR-010).

## Voice Session REST

### POST /sessions

Start a new voice session.

**Response 200:**
```json
{
  "session_id": "uuid",
  "websocket_url": "ws://host/voice/{session_id}",
  "expires_at": "2026-01-01T00:05:00Z"
}
```

## State Snapshot

`WillowAgent.get_session_state()` returns a `SessionSnapshot`:

```python
@dataclass
class SessionSnapshot:
    session_id: str
    current_m: float
    turn_count: int
    cold_start_active: bool
    troll_defense_active: bool
    residual_plot_weighted_avg: float
    last_updated: datetime
    tier_latencies: dict[str, float]
```

## ThoughtSignature

Generated by Tier 3 for each turn:

```python
@dataclass
class ThoughtSignature:
    intent: IntentType        # collaborative | insightful | neutral | hostile | devaluing | sincere_pivot
    tone: ToneType            # formal | casual | warm | aggressive | sarcastic
    detected_tactic: TacticType | None
    m_modifier: float
    tier_trigger: int         # 2 | 3 | 4
    rationale: str
```
