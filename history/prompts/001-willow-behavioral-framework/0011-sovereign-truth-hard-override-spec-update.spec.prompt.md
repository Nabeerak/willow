## Prompt

update specs Sovereign Truths must be stored as structured JSON and evaluated deterministically by a Python function before LLM response generation. The LLM must not receive contradicted Sovereign Truths as context — Tier 4 response must be programmatically constructed around the ground truth, bypassing LLM generation entirely for that assertion. This means T010-T013 in the Foundational phase need to be updated: the SovereignTruthCache is not just an LRU cache for lookup speed, it is a hard override layer that intercepts user input before the prompt is built. A deterministic contradiction detection function checks user input against cached truths first — if a match is found, Tier 4 fires and the response is constructed in Python, not generated by the model. The JSON lookup approach is chosen over vector embedding for MVP — fast, deterministic, zero hallucination risk. Vector embedding with similarity thresholds is deferred to a future version.
